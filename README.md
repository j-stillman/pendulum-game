# pendulum-game
A final project for my high school APCS class.

This project was written in Java with the Eclipse IDE. It has helped me solidify my understanding of OOP concepts. 

My inspiration for this project was "actually using the math you learn in school". I liked learning trigonometry in Algebra 2 so I made it a point to use the trig functions like sin() and cos(), so I made a system of pendulums that rotate in circles and draw their paths.

While a lot of the code could be written smarter/cleaner, I feel that this project demonstrates my capacity to learn new techniques and libraries. One day I plan on re-creating this project with skills that I've learned since, and I am considering doing just that for my software design class.
